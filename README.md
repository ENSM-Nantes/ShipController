# ShipController
Ship controller is a maritime console for controlling ships. It can be used with maritime simulators or for controlling automated ships.

![Bridge Command SOMOS project](https://github.com/ENSM-Nantes/ShipController/blob/main/examples/console%2Bbc.png)

You can test it with our version of Bridge Command: https://github.com/ENSM-Nantes/bc

The [wiki](https://github.com/ENSM-Nantes/ShipController/wiki/) integrates all the needed explanations.

This is a live project that will be improved through time.

Don't hesitate to collaborate!

# Dependencies

For using the Arduino Code, you need to install first Arduino Joystick Library.
https://github.com/MHeironimus/ArduinoJoystickLibrary/
