# ShipController
Ship controller is a maritime console for controlling ships. It can be used with maritime simulators or for controlling automated ships.

We tested it with our version of Bridge Command: https://github.com/ENSM-Nantes/bc

# Dependencies

For using the Arduino Code, you need to install first Arduino Joystick Library.
https://github.com/MHeironimus/ArduinoJoystickLibrary/

# Configuration

To be sur that you are using the good numbers for the buttons, the best is to use the testing tools to identify the buttons. For potentiometers, you need to follow this list:

X is Channel 1
Y is Channel 2
Z is Channel 3
